# Contributing to farmtown-bot

## Getting Started
1. Fork the repository
2. Create a feature branch from `dev`
3. Make your changes
4. Test thoroughly
5. Submit a pull request to `dev`

## Guidelines
- Use descriptive commit messages
- Follow existing code style
- Test with a single wallet before multi-wallet
- Never commit secrets or API keys

## Development
```bash
# Clone
git clone https://github.com/xbyteid/farmtown-bot.git
cd farmtown-bot

# Run single wallet
python3 farmtown-bot.py w01
```

## Security
- Report vulnerabilities privately via DM
- Never share API keys or wallet credentials
