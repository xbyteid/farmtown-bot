# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **DO NOT** open a public GitHub issue
2. DM the maintainer directly via GitHub or Twitter
3. Include a description of the vulnerability and steps to reproduce

## Response Timeline

- Initial response: within 48 hours
- Fix timeline: depends on severity

## Scope

This is a game automation tool. While it handles authentication tokens and proxy connections, it does **not** handle financial transactions or wallet seed phrases.

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅ |
| < 18.0  | ❌ |
